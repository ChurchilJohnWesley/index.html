COMPUTER BASICS

Basics of Computers


1. What is a C omputer?
A computer is an electronic device that processes data,
performing tasks according to a set of instructions called a program. It can store, retrieve,
and manipulate data.

Components of a Computer:


-Hardware: The physical parts of a computer.
-Software: The programs and applications that run on the hardware.

2. Types of Computers


-Desktop Computers: Designed for regular use at a single location.
-Laptops: Portable computers with built-in screens and keyboards.
-Tablets: Touchscreen devices that are highly portable.
-Servers: Powerful computers that provide data and resources to other computers.
3. Basic Hardware Components

a. Central Processing Unit (CPU)
- The brain of the computer where most calculations take place.
- Measured in gigahertz (GHz), indicating how many instructions it can process per second.
b. Memory (RAM)
- Temporary storage that the CPU uses to hold data for currently running programs.
- More RAM allows for more applications to run simultaneously.
c. Storage Devices
-Hard Disk Drive (HDD): Traditional storage device that uses spinning disks.
-Solid State Drive (SSD): Faster and more reliable storage that uses flash memory.
d. Motherboard
- The main circuit board that connects all components of the computer.
e. Input Devices
- Devices that allow users to input data, such as:
-Keyboard
-Mouse
-Scanner
f. Output Devices
- Devices that output data from the computer, such as:
-Monitor
-Printer
-Speakers
4. Software Overview

a. Operating System (OS)
- The software that manages hardware and software resources. Examples include:
-Windows
-macOS
-Linux
b. Applications
- Programs designed for end-users, such as:
-Web browser (e.g., Chrome, Firefox)
-Office suites (e.g., Microsoft Office, Google Workspace)
-Media players (e.g., VLC, iTunes)
5. Networking Basics
-Local Area Network (LAN): A network that connects computers within a small area.
-Wide Area Network (WAN): A network that covers a broad area, often using leased telecommunication lines.
-Internet: A global network of computers that communicates through protocols like TCP/IP.
6. Security Measures
-Antivirus Software: Protects against malware and viruses.
-Firewalls: Monitor incoming and outgoing network traffic based on security rules.
-Regular Updates: Keeping software and operating systems up-to-date to protect against vulnerabilities.
7. Basic Maintenance Tips
-Regularly update software to ensure security and performance.
-Clean hardware to prevent dust build-up.
-Backup data regularly to prevent loss.
Common Terms
-Byte: A unit of digital information (8 bits).
-Cloud Computing: Using remote servers hosted on the internet to store and manage data.
-Firmware: Specialized software that is embedded in hardware.
